
# 1.一个故事 （摘自·郝广才《今天》）
----------

> **《被同党刺杀的总统》**

----------

 政治的理想是举贤任能，而现实却是用人唯亲。
 
 古代是一朝天子一朝臣，到现代的民主政治，还是摆脱不了分赃、分肥，依然会一人得道，鸡犬升天。不要小看鸡和犬，它们并不低看自己，还感觉自己功劳最大，它们如果欲求不满，会反目成仇咬死主人。
 
 **美国第二十任总统詹姆斯·艾布拉姆·加菲尔德**上任前，政坛弥漫着腐败的风气，政客一旦选上，立刻论功分赃，把能掌握的职位分派给选举出钱出力的人。买官、卖官是半公开的交易，利益交换是江湖第一法则。
 
 加菲尔德是一名牧师，也是美国第一个有神职身份的总统。他给自己的使命是要推动公务员的改革，杜绝贪腐。因此，他上任后坚持不肯照惯例“政党分肥”，他要不分党派，用人唯贤。许多与他同党的共和党人，没有得到预期的职位，对他心怀不满。 
 
 加菲尔德在选举前就已经表明态度，但共和党人以为这是选举口号。改革是说给人民听的，只要选上，分肥才是给政客用的。没想到加菲尔德来真的，要跟他自己说的一样做，那他们出钱出力不是白费了吗？鸡犬不能升天，就要捣乱，闹个鸡犬不宁。

----------


 
 **查尔斯·吉托**就是其中一只鸡犬，同样也是牧师、作家、律师。他为加菲尔德写过一些文宣、讲稿，自认加菲尔德能当选总统，他的功劳很大，所以应该分块肉。他想要当大使，法国最好，瑞士也可以。
 
 他跟其他鸡犬一样，每天到国务院去排队，等着分肉。最后国务卿詹姆斯·吉莱斯皮·布莱恩亲口对他说，不要妄想求官，最好死了这条心，新政府不会满足他的要求。吉托很生气，气得想杀人……

----------


 
 **1881年7月2日**，加菲尔德和夫人准备去度假，在华盛顿火车站被一个家伙从背后连开两枪，紧急送医后，虽然暂时抢救回来，但不到三个月就因伤重去世。他是继林肯之后第二位被刺杀的总统。凶手正是求官不成心怀怨恨的吉托。其实加菲尔德并不知道吉托来求官，吉托也不知道加菲尔德不知道，就这样打死了一个可能是好总统的改革者。吉托被抓时，还大喊：“现在阿瑟是总统啦！”
 
 这个阿瑟就是切斯特·阿瑟，他是当时的副总统，顺理成章成为总统。阿瑟跟吉托一样是共和党里的保守派，也就是赞成大家来分肥的那一派。所以吉托觉得自己打死加菲尔德，虽然有罪跑不掉，但终究“分肥派”胜利了，**他牺牲自己，至少证明改革是空想，自肥分肥才是真理。**
 
 阿瑟过去当纽约的海关长官时，因不遵守公务员行政中立的命令，被前任的海斯总统开除，所以在共和党人眼中，他是公务员改革的受害者。照理他一上台，大家又可以回到分赃的美好日子。 
 
 没想到，阿瑟接任总统后，居然不计过去的私仇，摇身一变成为改革派，而且比加菲尔德更激进。美国历史上最重要的“彭德尔顿法案”就是在他手里通过的。这个法案确立公务员的录用、升迁，完全要以能力和绩效为依据，公务员不得因政党归属或政治原因而被免职，强力遏止党同伐异、大家分肥的歪风。

----------


 
 阿瑟是良心发现？还是有其他原因？
 
 加菲尔德出殡时，华盛顿有七万多人沿街目送灵车，他的家乡俄亥俄州的克利夫兰，有十五万人参加追悼，可见加菲尔德的改革立场深得人心。再加上他被“分肥派”刺杀，更激起人民的义愤。**阿瑟现在成为总统，他能不顺应民意吗？**
 
 不管怎样，加菲尔德的被刺，**直接推进了美国吏治的改革**。从历史的角度来看，是很值得。
 
 **他的被刺还帮助了另一项技术的进步，就是空调**。他重伤时，子弹卡在身体里取不出来。当时华盛顿正值七月炎夏，医生怕天热导致伤口发炎，海军一位工程师就想办法在白宫安装了世界上第一台空调，把室温降到十一摄氏度，但加菲尔德还是感染发烧，后来转到新泽西州的海边静养，依然抗不住发炎，9月19日过世，只有五十岁。
 
 为什么一般人对他很陌生？因为他是第二个被刺的美国总统。**第二个人是谁，往往被世人忽略。**

# 2.一个认知偏差（摘自维基百科）
----------

**后见之明偏误/Hindsight bias：**

> **又称“我早就知道了”、“马后炮”、“事后诸葛”。在事情发生或发展后，以为自己事前就能预测其发生与发展。**

地点：美国

鸡犬C:

> "膜拜大佬……请问……你是怎么看出来的？”
